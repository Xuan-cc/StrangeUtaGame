# UI 层设计

## 概述

UI 层基于 **PyQt6 + PyQt-Fluent-Widgets**，采用 `MSFluentWindow` 侧边栏导航架构（参考 March7thAssistant）。所有业务操作委托给应用服务层。

## 技术选型

| 组件 | 选择 | 理由 |
|------|------|------|
| GUI 框架 | PyQt6 | 成熟稳定，功能完善 |
| 组件库 | PyQt-Fluent-Widgets | 现代化 Fluent Design 风格 |
| 窗口架构 | MSFluentWindow | 侧边栏导航，现代化体验 |
| 样式方案 | QSS | 类似 CSS，易于维护 |

## 界面结构

```
MainWindow (MSFluentWindow)
├── 侧边栏导航
│   ├── 主页 (HomeInterface)
│   ├── 打轴 (EditorInterface) ← 核心界面
│   ├── 导出 (ExportInterface)
│   ├── 演唱者 (SingerManagerInterface)
│   ├── 注音编辑 (RubyInterface)
│   └── 设置 (SettingsInterface) [底部]
│
├── HomeInterface (主页)
│   ├── 新建项目卡片 (SimpleCardWidget)
│   │   ├── 歌词 TextEdit（支持粘贴、LRC/TXT/KRA）
│   │   ├── 导入歌词文件 / 清空
│   │   ├── 音频文件选择
│   │   └── 创建项目按钮
│   └── 打开项目卡片
│       └── 打开 .sug 项目按钮
│
├── EditorInterface (打轴界面)
│   ├── EditorToolBar (工具栏)
│   │   ├── 保存项目 / 加载音频
│   │   ├── 撤销 / 重做
│   │   └── 音频文件名显示
│   ├── TransportBar (播放控制栏)
│   │   ├── 停止 / 播放·暂停
│   │   ├── 时间显示 (MM:SS.CC)
│   │   ├── 进度条 / 速度 / 音量
│   ├── TimelineWidget (时间轴)
│   ├── KaraokePreview (歌词预览, ±3 行上下文)
│   ├── 底部打轴操作栏
│   │   ├── 当前行信息 / 打轴按钮 / 清除标签
│   │   └── 快捷键提示
│   └── 状态栏 (播放状态 + 打轴进度)
│
├── ExportInterface (导出)
│   ├── 格式列表 (LRC/KRA/TXT/ASS/Nicokara)
│   ├── 输出路径选择
│   ├── 单格式导出 / 批量导出
│
├── SingerManagerInterface (演唱者管理)
│   ├── 演唱者列表 ([后台编号] 显示名)
│   └── 添加 / 编辑 / 删除
│
├── RubyInterface (注音编辑)
│   ├── 行选择导航
│   ├── 注音表格
│   └── 自动分析 / 添加注音
│
└── SettingsInterface (设置)
    ├── 音频 / 打轴 / 界面 / 导出 / 快捷键设置
    └── 关于信息
```

## 快捷键

| 按键 | 功能 | 状态 |
|------|------|------|
| **Space** | 打轴（记录时间标签） | ✅ |
| **A** | 播放/暂停 | ✅ |
| **S** | 停止 | ✅ |
| **Z** | 后退 5 秒 | ✅ |
| **X** | 前进 5 秒 | ✅ |
| **Q** | 减速 0.1x | ✅ |
| **W** | 加速 0.1x | ✅ |
| **Esc** | 清除当前行标签 | ✅ |

### 计划中

| 按键 | 功能 |
|------|------|
| **Space 长按** | 句尾拖音（按下开始，抬起结束） |
| **+/-** | 添加/删除 Checkpoint |
| **L** | 切换句尾标记 |
| **F1–F9** | 可配置功能键 |
| **双击 Checkpoint** | 预听（跳转到该时间前 X 秒） |
| **Alt+滚轮** | 缩放歌词显示 |

## 窗口行为

- 关闭按钮(X)：退出应用（释放音频资源）
- 默认尺寸：1400×900，最小：1200×800
- 启动时居中
