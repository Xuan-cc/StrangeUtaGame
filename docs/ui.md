# UI 层设计

StrangeUtaGame 提供了三套主要的编辑界面，分别针对打轴、编辑和注音任务。

## 核心界面

### 1. EditorInterface (打轴界面)
这是主要的时间标签编辑界面。

- **职责**：
    - 提供类似节奏游戏的打轴体验。
    - 实时预览卡拉OK走字效果 (KaraokePreview)。
    - 全键盘驱动，支持高频率的时间标签记录。
    - **核心特性**：
        - 逐字高亮渲染，支持 Ruby 同步走字。
        - 连词 (Linked Words) 渲染。
        - 实时应用时间偏移。

### 2. EditInterface (编辑界面)
提供基于表格的项目全局视图。

- **职责**：
    - 展示所有歌词行及其元数据（行号、演唱者、连词标记等）。
    - 提供批量操作入口。
    - **LineDetailDialog**：双击行打开详细编辑对话框，可逐个 Character 编辑注音、节奏点和演唱者。

### 3. RubyInterface (注音界面)
专注于注音编辑和文本调整。

- **职责**：
    - 自动分析汉字假名注音。
    - 手动编辑注音文本及其对应的字符范围。
    - 支持按字符类型批量清理注音。
    - **自由文本编辑**：支持直接修改歌词行文本，系统自动维护已有的时间标签和注音。

## 辅助界面
- **MainWindow**：基于 FluentWindow 的主框架，包含侧边栏导航。
- **StartupInterface**：项目创建向导，负责初始歌词导入和音频选择。
- **HomeInterface**：主页，管理最近音频和快捷入口。
