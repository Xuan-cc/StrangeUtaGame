# UI 层设计

## 概述

UI 层基于 **PyQt6 + PyQt-Fluent-Widgets**，采用 `MSFluentWindow` 侧边栏导航架构（参考 March7thAssistant）。所有业务操作委托给应用服务层。

## 技术选型

| 组件 | 选择 | 理由 |
|------|------|------|
| GUI 框架 | PyQt6 | 成熟稳定，功能完善 |
| 组件库 | PyQt-Fluent-Widgets | 现代化 Fluent Design 风格 |
| 窗口架构 | MSFluentWindow | 侧边栏导航，现代化体验 |
| 样式方案 | QSS | 类似 CSS，易于维护 |

### 快捷键系统特性

- **键盘监听捕获**：点击快捷键设置按钮后，直接按键即可设定，支持组合键（Ctrl+X、Alt+A 等）
- **双快捷键绑定**：每个功能可设置两个快捷键（主键 + 备用键）
- **冲突检测**：保存时自动检测按键冲突，被冲突方的按键自动清除并提示用户
- **ESC 取消**：设置快捷键时按 ESC 可取消当前设置操作，不做任何改动
- **所有快捷键均可在设置界面自定义**
